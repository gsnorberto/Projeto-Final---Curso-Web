INSTALAÇÃO DO POSTGRES

COMANDOS
    - npm start (inciar)
    - npm i -g knex  (instalação do knex - que é uma biblioteca que trabalha com vários bancos de dados, e nesse projeto ele será usado para acessar o postgre).
    - npx knex init (Cria o arquivo knexfile)
    - knex migrate:make create_table_users (Cria uma tabela de usuários)
    - knex migrate:make create_table_categories
    - knex migrate:make create_table_articles


ACESSO
    - http://localhost:3000/users
